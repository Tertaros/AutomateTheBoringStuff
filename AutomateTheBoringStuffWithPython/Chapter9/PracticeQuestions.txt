1. What is a relative path relative to?
They are relative to the current working directory

2. What does an absolute path start with?
The anchor/ root which is usually on Windows the drive C: and on Linux and Mac "/"

3. What does Path('C:/Users') / 'Al' evaluate to on Windows?
C:/Users/Al

4. What does 'C:/Users' / 'Al' evaluate to on Windows?
It will result in an error, at least one of the initial two variables needs to be of type path to use the "/" to concatenate a path.

5. What do the os.getcwd() and os.chdir() functions do?
os.getcwd() will retrieve the current working directory.
os.chdir() will change the current working directoy.

6. What are the . and .. folders?
. is the current folder and .. the parent folder.

7. In C:\bacon\eggs\spam.txt, which part is the dir name, and which part is the base name?
dir name = C:\bacon\eggs
base name = spam.txt

8. What are the three “mode” arguments that can be passed to the open() function?
r-> read
w-> write
a-> append

9. What happens if an existing file is opened in write mode?
It's content will be overwritten.

10. What is the difference between the read() and readlines() methods?
read() will return the entire content of the file as a single string.
readlines() will return a list of strings that is split based on line breaks of the read file.

11. What data structure does a shelf value resemble?
A dictionary as it uses both keys and values.
