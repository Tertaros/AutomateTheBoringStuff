4. What is an expression made up of? What do all expressions do?
They are made up of values. They always result in a single value.

5. This chapter introduced assignment statements, like spam = 10. What is the difference between an expression and a statement?

8. Why is eggs a valid variable name while 100 is invalid?
variables are not allowed to be purely a number or start with a number, they can contain text though.

9. What three functions can be used to get the integer, floating-point number or string version of a value?
int(), float(), str()

10. Why does this expression cause an error? How can you fix it?
'I have eaten ' + 99 + ' burritos.'
The errors comes from trying to add an integer to a string, this can be resolved by converting the int with str() to a string before the addition
'I have eaten ' + str(99) + ' burritos.'
